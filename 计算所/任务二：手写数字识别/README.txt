手写数字识别任务介绍：

这次的任务主要有三个，1.生成数据集，每个人手写300个数字并扫描进电脑。2.构建利用mnist数据集和自己3000个数据集训练一个自己的网络，要求识别出数字的类别。3.利用自己3000个数据集训练一个识别出写数字的人（共5人）的网络，然后再用迁移学习（利用前一个数字种类识别网络模型）的方式再次训练。

文件介绍：

扫描数据：里面是原手机扫描的图片以及生成的数据及其代码。

1_数字识别代码.ipynb : 10个数字识别的代码

1_数字识别报告.ipynb : 此任务的报告

2_数字作者识别以及迁移学习.ipynb : 5个数字作者识别代码及其报告，迁移学习代码

model_num.h5 : 识别数字种类的网络模型。

model_per0.h5 : 识别数字作者的网络模型。

x_train.npy : （3000, 28, 28）的numpy文件，存储所有图片信息

y_train.npy :   (3000,) 的numpy文件，存储所有对应图片种类信息

其余图片文件均为报告展示的文件
